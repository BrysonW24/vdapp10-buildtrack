### The Idea

Construction projects live in spreadsheets, WhatsApp groups, and filing cabinets. When something goes wrong, good luck finding the photo that proves what happened. BuildTrack puts everything in one place — progress photos, tasks, documents, and team communication. No more "who has the latest plans?"

---

### What It Does

| | |
|---|---|
| **Project Dashboard** | Overview of all active projects with progress indicators |
| **Photo Documentation** | Capture and organize site photos with annotations |
| **Task Management** | Assign and track work items with status badges |
| **Document Hub** | Plans, permits, contracts — all in one place |
| **Team Chat** | Real-time communication with crews |
| **Timeline** | Gantt chart view of project phases |
